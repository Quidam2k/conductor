What is this?
Conductor coordinates groups of people using their phones. An organizer creates a timed sequence of cues — spoken announcements, vibrations, or audio clips that play at specific times — and shares the event as a link or QR code. Every participant's device plays the same cues at the same time.

Tap Load Demo Event below to try it right now — no downloads or setup needed. The app uses your phone's text-to-speech to play each cue.

Events can be password-protected with AES-256 encryption. No data is ever sent to a server — everything stays on your device. No accounts, no app store, no server required.

Designed for peaceful coordination — performances, community events, demonstrations. No telemetry, no tracking. By design.

Tip: For live events, disable battery saver and keep your screen on.

How to Create Events
Use the built-in editor: Tap Create New Event below. Give it a title, start time, and description, then add actions with timestamps. The editor walks you through it step by step.

Write a text file: Create a .txt file with timestamps and actions — one per line (e.g. 0:30 Wave left). Add optional tags like [emphasis] for gold highlighting, [countdown] for an audible countdown, or [alert] for urgent red cues. Paste the text into the input field below or import the file.

Start from the demo: Tap Load Demo Event to see how a finished event looks. You can study the format and adapt it for your own event.
