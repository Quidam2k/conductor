## Quick Start (5 minutes)

### Step 1: Start the APK Server
```bash
cd devs/conductor/conductor-mobile/scripts
.\serve-apk.bat
```
This starts a web server on your PC. You'll see your local IP address (like `192.168.1.x`).

### Step 2: Install on Your Pixel
1. On your phone, open Chrome
2. Go to `http://YOUR_PC_IP:8000`
3. Tap the APK link
4. Allow installation from unknown sources if prompted
5. Install takes ~10 seconds

### Step 3: Generate Fresh QR Code
```bash
python devs/conductor/test-data/generate-pantheon-qr.py
```
This creates `pantheon-inaugural-qr.png` with an event starting 2 minutes from now.

### Step 4: Launch & Scan
1. Open the Conductor app
2. Tap "Scan QR Code"
3. Point your camera at the QR code PNG on your monitor
4. The event loads instantly (no internet needed!)

### Step 5: Go Live
1. Review the timeline preview
2. Tap "Go Live" when ready
3. The circular timeline shows your progress
4. Listen for TTS announcements
5. Feel the haptic patterns

## Troubleshooting

**QR code won't scan:**
- Make sure the PNG is displayed clearly (no reflections)
- Hold phone steady about 6 inches from screen
- Try increasing screen brightness

**No audio:**
- Check phone volume
- The app uses TTS (text-to-speech)
- Some actions may not have audio if TTS fails

**No haptics:**
- Make sure vibration is enabled in phone settings
- Some phones have vibration intensity settings

**Event shows "expired":**
- The QR encodes a specific start time
- Regenerate with the Python script for a fresh time

## What's Happening Under the Hood

1. **QR contains the entire event** - No server needed
2. **Data is gzip compressed** - Fits in ~600 character URL
3. **Deep linking** - `conductor://event/...` opens directly in app
4. **System alarms** - Even if you kill the app, notifications fire
5. **Offline-first** - Works in airplane mode after scanning
